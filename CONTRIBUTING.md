# Guidelines pour contribuer

Merci de vouloir contribuer ! Pour participer :

1. Fork le projet.
2. Crée une branche pour ta modification (`git checkout -b feature/ma-modification`).
3. Commit tes changements (`git commit -m "Ajout de fonctionnalité"`).
4. Push sur ta branche (`git push origin feature/ma-modification`).
5. Ouvre une Pull Request.
